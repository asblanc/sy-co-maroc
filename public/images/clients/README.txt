Logos officiels des clients / partenaires
=========================================

Déposez ici les fichiers logos (PNG/SVG/JPG à fond transparent de préférence),
puis renseignez le chemin dans lib/data.ts → `partners` :

  { name: "OCP Group", logo: "/images/clients/ocp.png" },

Sans `logo`, une tuile texte stylée est affichée automatiquement.
N'utilisez que les logos d'entreprises réellement clientes / partenaires,
avec leur autorisation.
